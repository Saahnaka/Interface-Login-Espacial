# 🌌 Interface de Login e Cadastro Espacial

Uma interface de autenticação moderna, responsiva e com temática espacial, desenvolvida para simular telas de login e cadastro com transição de estados e design imersivo.

## 🎯 Sobre o Projeto
Este projeto foi construído com foco em **UI/UX (User Interface e User Experience)**, unindo um visual limpo e escuro com elementos espaciais dinâmicos. A aplicação simula o fluxo completo de um usuário: acesso à tela de login, alternância para a criação de conta (registro), validação de campos e feedback visual de sucesso na autenticação.

## 🚀 Funcionalidades
- **Alternância de Telas:** Transição fluida entre a tela de "Login" e a tela de "Criar Conta".
- **Campos de Autenticação:** Inputs estilizados para e-mail, senha e confirmação de senha.
- **Opções de Interação:** Caixa de seleção "Lembrar de mim", botão "Esqueci minha senha" e mensagens interativas de redirecionamento.
- **Simulação de Sistema:** Telas de alerta/feedback dinâmicas para sucesso no registro e simulação de login realizado.

## 🛠️ Tecnologias Utilizadas
- **HTML5** (Estruturação semântica)
- **CSS3** (Estilização avançada, layout centralizado e tema espacial)
